## VibeMS8 Music Player

A simple UI to play music stored in the "music folder" using the HTML5 audio API

## Features

- UI for music player including spinning image and song detail popup
- Add play and pause functionality
- Switch songs
- Progress bar


## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
